## 🐛 常见问题排查

### 图片不显示
**问题**: 卡片或装饰图片显示为破损图标
**解决**:
1. 检查图片文件是否存在于 `public/` 目录
2. 检查图片路径是否正确
3. 清除浏览器缓存后重试

### 装饰位置错乱
**问题**: 装饰贴纸位置不正确
**解决**:
1. 检查 CSS 绝对定位是否正确
2. 查看响应式断点是否生效
3. 检查 z-index 层级关系

### 动画卡顿
**问题**: 浮动动画不流畅
**解决**:
1. 减少同时浮动的元素数量
2. 检查是否有大量DOM操作
3. 考虑使用 CSS transform 而非 position

### 组件导入错误
**问题**: React 组件无法导入
**解决**:
1. 检查组件文件路径是否正确
2. 确认组件是否正确导出 (export default)
3. 检查 import 语句是否正确
